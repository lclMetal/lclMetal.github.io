Hei,

olen Lassi Lehtinen, -vuotias ohjelmistoinsinööri Tampereelta. Opiskelin tieto- ja viestintätekniikkaa JAMKissa Jyväskylässä, keskittyen ohjelmistotekniikkaan. Valmistuin kesällä 2023.

Olen harrastanut ohjelmointia yläkouluikäisestä asti. Tuolloin tutustuin Game Editor -nimiseen pelinteko-ohjelmaan, joka käyttää C-ohjelmointikieltä. Kiinnostuin ohjelmoinnista nopeasti, ja huomasin pian olevani enemmän innoissani erilaisten pelimekaniikkojen ohjelmoinnin tarjoamista haasteista kuin pelien tekemisestä.

Game Editorin parissa opin paljon C-ohjelmoinnista. Myöhemmin olen ohjelmoinut myös seuraavilla ohjelmointikielillä: C#, Java, C++, JavaScript, PHP ja Elixir. Ohjelmoinnin lisäksi hallitsen HTML- ja CSS-kuvauskielten sujuvan käytön.

Tällä sivustolla on näytteitä töistäni niin ohjelmoinnin, web-suunnittelun, kuin grafiikankin osalta.

Hello,

I'm Lassi Lehtinen, a -year-old software engineer from Tampere. I studied ICT at JAMK, Jyväskylä with a focus on software engineering. I received my bachelor's degree in summer 2023.

I've programmed ever since I was 14 years old. That's when I found Game Editor, a game creation software that uses the C programming language for scripting game logic and functionality. I quickly got interested in programming and soon noticed myself being more interested in conquering the programming challenges implementing different game mechanics offered than actually making finished games.

In my time working with Game Editor I learned a lot about C programming. Later on I've also programmed with the following programming languages: C#, Java, C++, JavaScript, PHP and Elixir. In addition to programming, I am fluent with the HTML and CSS markup languages.

This portfolio site presents samples of my work in programming, web design and graphics.
